# 基于vue3-vite-Element搭建的后台管理系統

### Build Setup
install dependencies
> npm install

Compiles and hot-reloads for development
> npm run dev

Compiles and minifies for production
> npm run build

- 用新的公共組件table重寫Guest,Good,Swiper,以減少商品管理，輪播圖配置，會員管理部分的代碼量
